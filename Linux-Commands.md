lsblk	Показывает все диски и разделы деревом.	Всегда, чтобы понять, что есть.
lsblk -f	То же + файловая система (ext4/ntfs/swap) и UUID.	Перед монтированием или форматированием.
lsblk -o NAME,SIZE,MODEL	Имя, размер, модель.	Чтобы точно опознать внешний диск по названию (WDC, Samsung и т.д.).
sudo fdisk -l	Детальная инфа о разделах (тип, сектора).	Если нужно больше, чем даёт lsblk.
sudo blkid	UUID и тип ФС для всех разделов.	Для fstab или чтобы скопировать UUID.
df -h	Показывает смонтированные разделы и свободное место.	Проверить, что примонтировалось и не забито ли.
sudo dmesg | grep -i usb	Логи по USB.	Если диск то появляется, то пропадает.
sudo dmesg -w	Следить за логами в реальном времени.	Воткнуть диск и 
sudo fdisk /dev/sdX	Запуск редактора разделов (MBR). Внутри: p — показать, d — удалить, n — новый, w — записать.	Не забудь w, иначе изменения не сохранятся.
sudo gdisk /dev/sdX	То же, но для GPT (современные UEFI-системы). Команды те же.	Если не знаешь, что ставить — ставь GPT.
sudo mkfs.ext4 /dev/sdX1	Форматирует раздел /dev/sdX1 в ext4.	Все данные на разделе сгорят.
sudo mkswap /dev/sdX2	Делает раздел swap-областью.	Потом надо включить: swapon.
sudo swapon /dev/sdX2	Включает swap.	Проверить: swapon --show.
sudo wipefs -a /dev/sdX	Стирает ВСЕ файловые системы и таблицу разделов с диска.	Полная зачистка диска. Данные не восстановить.
sudo dd if=/dev/zero of=/dev/sdX bs=512 count=1	Затирает первый сектор (убивает таблицу разделов).	Мягче, чем wipefs. Тоже необратимо.
