## 🧱 Основы

### Типы данных

| Тип | Пример | Что делает |
|-----|--------|------------|
| `int` | `42` | Целое число |
| `float` | `3.14` | Дробное число |
| `str` | `"hello"` | Строка (текст) |
| `bool` | `True` / `False` | Логическое значение |
| `list` | `[1, 2, 3]` | Изменяемый список |
| `tuple` | `(1, 2, 3)` | Неизменяемый список |
| `dict` | `{"key": "value"}` | Словарь (ключ → значение) |
| `None` | `None` | Отсутствие значения |

### Переменные и операции

```python
name = "Андерсон"
count = 42
is_ready = True

# Арифметика
sum = count + 10           # 52
diff = count - 10          # 32
prod = count * 2           # 84
div = count / 2            # 21.0 (всегда float)
int_div = count // 2       # 21 (целочисленное деление)
rem = count % 3            # 0 (остаток от деления)

# Работа со строками
greeting = f"Привет, {name}!"   # f-строка (вставляет переменные)
len(name)                        # длина строки
name.upper()                     # "АНДЕРСОН"
