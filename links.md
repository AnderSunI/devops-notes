# Полезные ссылки

Сборник ресурсов, которые пригодились в работе и учёбе. Постоянно пополняется.

---

## 🐧 Linux

| Ресурс | Описание |
|--------|----------|
| [Linux Journey](https://linuxjourney.com/) | Интерактивный курс от основ до ядра, сетей и скриптов. |
| [The Linux Command Line](https://linuxcommand.org/tlcl.php) | Книга Уильяма Шоттса (бесплатный PDF). Основа для работы в терминале. |
| [Bash Hackers Wiki](https://wiki.bash-hackers.org/) | Глубокое погружение в bash, тонны примеров и подводных камней. |
| [DigitalOcean Linux Tutorials](https://www.digitalocean.com/community/tags/linux) | Практические гайды по настройке серверов, сетей, безопасности. |
| [Arch Wiki](https://wiki.archlinux.org/) | Лучшая документация по Linux вообще. Даже если не используешь Arch, здесь находят решения. |
| [Explain Shell](https://explainshell.com/) | Вставляешь команду — получаешь разбор каждого аргумента. |
| [Ubuntu Server Guide](https://ubuntu.com/server/docs) | Официальная документация по Ubuntu Server (наша основная ОС). |

## 🪟 Windows (администрирование, PowerShell)

| Ресурс | Описание |
|--------|----------|
| [Microsoft Learn: Windows Server](https://learn.microsoft.com/en-us/windows-server/) | Официальная документация: AD, DNS, DHCP, групповые политики. |
| [PowerShell Documentation](https://learn.microsoft.com/en-us/powershell/) | Официальная справка. Включает советы по переходу с CMD. |
| [Adam the Automator](https://adamtheautomator.com/) | Блог с тысячами статей по PowerShell, автоматизации, AD, Azure. |
| [SS64 Windows CMD & PS](https://ss64.com/nt/) | Шпаргалки по командам Windows и PowerShell с примерами. |
| [Sysinternals Suite](https://learn.microsoft.com/en-us/sysinternals/) | Набор утилит от Марка Руссиновича. Без них ни один админ не обходится. |
